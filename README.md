## 💻 Local Development
If you want to run this site locally, ensure you have Hugo (extended version) installed.

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/Lourencom/Lourencom.github.io.git
    cd Lourencom.github.io
    ```
2.  **Run the local server:**
    ```bash
    hugo server
    ```
3.  **View the site:**
    Open `http://localhost:1313` in your browser.
